Flowtune Documentation

🚀 Overview

Flowtune is a DSL and SDK for optimized resource loading by size, priority, and strategy. Supports Python and JavaScript.

📐 Syntax Reference

  • resource: define resource properties
  • group: group multiple resources
  • run: define execution order
  • auto_config: apply preset security/optimization

🐍 Python SDK

pip install flowtune

from flowtune.parser import QuantumFlowtuneParser
from flowtune.executor import QuantumFlowtuneExecutor

parser = QuantumFlowtuneParser('example.ft')
parser.parse(auto_config=True)

executor = QuantumFlowtuneExecutor(parser.resources, parser.groups, parser.execution_plan)
executor.run()

🌐 JavaScript SDK

npm install flowtune-js

import FlowtuneJS from 'flowtune-js';

fetch('example.ft.json')
  .then(res => res.json())
  .then(data => {
    const flowtune = new FlowtuneJS(data, {
      allowedDomains: ['cdn.example.com'],
      autoConfig: {
        security: 'strict',
        optimization: 'speed'
      }
    });
    flowtune.run();
  });
